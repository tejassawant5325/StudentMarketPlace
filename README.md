# 🎓 Student Marketplace

A web application built using **ASP.NET MVC** where students can buy, sell, and share products within their college community. The project was created to make it easier for students to exchange study materials and second-hand items.

## Features

- Student Registration & Login
- Add, Edit, and Delete Product Listings
- Browse Products
- Digital and Physical Products
- Seller Dashboard
- Contact Seller
- Contact Us Page
- Responsive Design

## Tech Stack

- ASP.NET MVC (C#)
- HTML
- CSS
- Bootstrap
- SQL Server
- Visual Studio

## Product Categories

### Digital
- Notes
- PDFs
- Study Materials

### Physical
- Books
- Stationery
- Electronics
- Other Student Essentials
